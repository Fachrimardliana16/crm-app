## ğŸ” Troubleshooting

### Problem: "2 Faktur tidak muat dalam 1 halaman"
**Solusi:**
- Pastikan browser zoom 100%
- Jangan gunakan "Fit to page" di print dialog
- Pilih margin **Default** (bukan minimum)
- Cek apakah paper size **A4** sudah benar

### Problem: "Faktur terpotong di tengah"
**Solusi:**
- Di browser print preview, pastikan **Portrait**
- Margin setting gunakan **Default** 
- Jangan gunakan **Custom margins**
- Pastikan printer driver mendukung A4 penuh

### Problem: "Font terlalu kecil saat print"
**Solusi:**
- Print dengan kualitas **High** atau **Best**
- Pastikan scale **100%** (bukan 85% atau fit to page)
- Jangan gunakan "Draft" mode printer

### Problem: "Gap terlalu kecil untuk potong"
**Solusi:**
- Gap standar 0.5cm sudah cukup untuk potong
- Gunakan paper cutter untuk hasil rapi
- Marking dengan pensil dulu sebelum potong
